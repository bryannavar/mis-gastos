¿Cómo usar Mi Presupuesto?
🏠 Inicio

Muestra el saldo del mes, ingresos, egresos y las últimas transacciones. Todo se consolida en la moneda principal del espacio.

➕ Agregar

Registrá ingresos y egresos. Si la transacción es en moneda extranjera y es de hoy, el tipo de cambio se obtiene automáticamente. Para fechas pasadas debés cargarlo manualmente en Config.

🔄 Fijos

Registrá gastos que se repiten cada mes (Netflix, alquiler, etc.). Se aplican automáticamente al abrir la app cada mes en el día que definiste.

🎯 Presupuesto

Definí cuánto querés gastar por categoría cada mes. La barra muestra cuánto llevás gastado. Podés importar el presupuesto del mes anterior con un clic.

📋 Historial

Consultá todas las transacciones de cualquier mes y el resumen del presupuesto de ese período.

⚙️ Config

Registrá el tipo de cambio manualmente para fechas pasadas. Podés editar o borrar registros existentes.

📁 Espacios

Creá espacios separados para tu vida personal, negocio o pyme. Cada espacio tiene sus propias transacciones, presupuestos y tipo de cambio. Usá el lápiz ✏️ para editar el espacio activo y el + para crear uno nuevo.

💱 Tipo de cambio

Para transacciones en dólares con fecha de hoy, el sistema obtiene el tipo de cambio automáticamente desde el Ministerio de Hacienda de Costa Rica. Para otras fechas o monedas, cargalo manualmente en Config.
